# CBC Rubric Generator 2025

A lightweight web-based tool designed for Kenyan primary school teachers to generate Competency-Based Curriculum (CBC) compliant assessment rubrics instantly.

## Features
- Follows KICD standard levels (Exceeding, Meeting, Approaching, Below Expectations).
- Generates descriptive text for rubrics based on user input.
- Print-ready format for school records.

## Installation
1. Install Node.js.
2. Run `npm install` to install dependencies.
3. Run `npm start` to launch the server.
4. Open `http://localhost:3000` in your browser.

## Usage
- Enter the Learning Area (Subject).
- Enter the specific Learning Outcome/Strand.
- Click Generate and Print.
